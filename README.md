## Enterprise Vehicle Operation Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/6be69908945a4b7a9abeff49770991a0/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Feature Description
The system is divided into (roles): regular users, drivers, administrators.

**Regular User Features:**
1. Login
2. Registration
3. Homepage
4. Drivers
5. Vehicle Registration
6. Enterprise Forum
7. System Announcements
8. Personal Center
9. My Posts

**Driver Features:**
1. Login
2. Registration
3. Change Password
4. Personal Center
5. Vehicle Registration Management
6. Maintenance Information Management
7. Accident Information Management
8. Violation Information Management
9. Vehicle Operation Management



**Administrator Features:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Driver Management
4. User Management
5. Vehicle Registration Management
6. Maintenance Information Management
7. Accident Information Management
8. Violation Information Management
9. Vehicle Operation Management
10. Enterprise Forum
11. System Management (System Announcements, About Us, System Introduction, Carousel Image Management)
